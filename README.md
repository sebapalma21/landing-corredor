# CostaAndes Propiedades (demo) — Landing inmobiliaria

Sitio 100% estático con datos ficticios (Quinta Región, Valparaíso).

## Archivos
- index.html
- styles.css
- app.js
- properties.json (catálogo ficticio)

## Deploy rápido (Netlify)
1) En Netlify: Add new site → Deploy manually
2) Arrastra la carpeta completa (o el zip descomprimido)
3) Listo: tendrás una URL tipo `algo.netlify.app`

### Formulario (Netlify Forms)
El formulario funciona automáticamente en Netlify.
Cada envío consume 1 crédito en el plan Free.

## Personalizar
- Cambia marca, contacto y propiedades en `properties.json`.
- Textos y secciones: `index.html`.
- Estilos: `styles.css`.
